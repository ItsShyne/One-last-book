These images are have multiple layers to them. I've provided both separated and combined versions of each image here.

The ones that are meant to be multi-layered are in separate folders named "[image]_layered".
These are meant to be displayed in the following order: bg -> optional bg element (if applicable) -> character of your choice -> fg element
Each folder also has a mockup that includes a character, just to give an idea of what it would look like.

If you don't want to use the layered backgrounds, then there are premade ones in the folder this readme's in.
If the image has an optional background element, then there will one version with it, and one version without it.
If you don't want any elements in the image at all, then use the bg from the image's "[image]_layered" folder.
