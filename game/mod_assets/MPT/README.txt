If you have just downloaded this and you're wondering why there's a folder named "LayeredRedux [MPT] - Family Friendly",
it's because I decided to make a version that removes all the Yuri sprites that feature blood, cuts and stabs. 

Choose whichever you think is more suitable for your project!

------------------------------------------------------------------------------

Instructions:

1. Install MPT to your Mod if you haven't already.

2. Place the "Yuri" folder in your MPT folder. 
[This will replace the original MPT Sprites]

==============================================================================================================
= I recommend you to make a backup of your original "yuri_layeredimage.rpy" file during the next step.     =
==============================================================================================================

3. [How to add the Extra Assets]

Depending on whether you have modified the original "yuri_layeredimage.rpy" file or not, 
you must do one of two things:
    
    A) If you have NOT modified the original file, then you can just drag the 
    "yuri_layeredimage.rpy" file from the LayeredRedux zip to your MPT folder and 
    click on "Replace the file in the destination".
    
    B) If you HAVE modified the original file, then you'll have to manually merge 
    the contents of them together.

-----------------------------------------------------------------------------------------------------------------


Notes:

-There's a folder included that's named "pdns". It has a bunch of Paint.NET project files of different assets present, 
allowing for further edits to be made. You can delete this folder if you want to.

-If you are installing LayeredRedux on a Mod with MPT already installed, you'll still have the missing assets available to use.
